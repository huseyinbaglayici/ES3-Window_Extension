# Changelog

## [1.0.0] - 2026-07-04

### Added
- Key Registry editor window (`Window > ES3 Key Manager`) for defining keys, primitive types, and default values.
- `ES3Keys` static runtime accessor with per-type Load/Save methods.
- `ES3KeyRegistry` ScriptableObject stored in `Resources`, auto-created by the window.
- Runtime and Editor assembly definitions.
